# Bayesian Sample Size Determination in Erlang Single-Server Queues

This repository contains the implementation and computational experiments related to the article:

**Laxmipriya Singh, E.S. Gomes, F.R.B. Cruz**  
*A Bayesian approach to determine sample size in Erlang single-server queues*  
Communications in Statistics - Theory and Methods, 2024  

DOI: https://doi.org/10.1080/03610926.2024.2401602

## Abstract

This repository provides the computational implementation supporting the numerical experiments presented in the article on Bayesian sample size determination for Erlang single-server queueing systems.

The study develops a Bayesian framework for determining appropriate sample sizes in queueing inference problems, with emphasis on efficient estimation of queue performance measures.

## Repository Structure

- `experiments.R`  
  Contains the code required to reproduce all numerical experiments presented in the article.

- `plots.R`  
  Reproduces all figures presented in the article.

- `results/`  
  Stores numerical outputs generated by the experiments.

- `figures/`  
  Stores generated plots and graphical outputs.

## Reproducibility

To reproduce the results reported in the paper:

1. Run:

```r
source("experiments.R")
```

2. Generate figures with:

```r
source("plots.R")
```
